# random20

crea 20 numeros random, 4 veces al dia.

[URL](https://t14fvby6ie.execute-api.us-east-1.amazonaws.com/dev/api/v0.1)

## options
[/all] (https://t14fvby6ie.execute-api.us-east-1.amazonaws.com/dev/api/v0.1/all) devuelve todos los numeros creados hasta la fecha
[/last] (https://t14fvby6ie.execute-api.us-east-1.amazonaws.com/dev/api/v0.1/last) devuelve los ultimos 20 numeros creados
[/date/:date] (https://t14fvby6ie.execute-api.us-east-1.amazonaws.com/dev/api/v0.1/date/:date) devuelve los 20 numeros creados en la fecha y hora seleccionada

## horarios

Los numeros se generan todos los dias a las 12:00, 15:00, 17:30, 21:00 UTC
